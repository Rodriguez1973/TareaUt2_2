Datos de Tipo String UNICODE

String

La clase String incluye métodos y propiedades para procesar cadenas de caracteres.

Un string es un array de caracteres, con un indice entre 0 y número_de_caracteres-1. Cada índice representa un carácter del string.

La propiedad length devuelve el número de caracteres en el string:
'ciudad'.length=>6

Utilizando el string como un array:
'ciudad'[2]=>'u'
Devuelve el caracter del índice 2 (comienza en 0).

Método:
'ciudad'.charCodeAt(2)=>117
Devuelve el código UNICODE del tercer carácter.

Método:
'ciudad'.indexOf('da')=>3
Devuelve la posición de la subcadena.

Método:
'ciudad'.substring(2, 5)=>'uda'
Devuelve la subcadena entre ambos índices.
